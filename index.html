General Notes

How I Work

  • Infrastructure-as-code first. Terraform modules built for reuse across environments, not one-off scripts.
  • GitOps over manual deploys. FluxCD reconciling from Git, not pipelines pushing state by hand.
  • Security as a design input, not a review-board rubber stamp — Keycloak, Sealed Secrets, IAM least-privilege from the start.
  • Handoff is part of the deliverable. Documentation and training so the team I worked with can run it without me.
